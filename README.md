# Spectrum 2.4 GHz

Una aplicación para ver la congestión de redes Wi-Fi en la frecuencia 2.4 GHz.

## Compilación

Primero, asegurese de instalar el [SDK de Android](https://developer.android.com/studio) (si utiliza el IDE la configuración es automática, pero con `commandlinetools` es suficiente). Luego configure la variable de entorno `ANDROID_HOME` para señalar la ubicación del SDK.

Una vez configurado el SDK, ejecute `./gradlew` en el directorio raíz para descargar todas las dependencias.

Puede ver todas las tareas disponibles ejecutando: `./gradlew tasks`.

Si desea compilar una version de depuración, ejecute:

```shell
./gradlew assembleDebug
```

Esto compilará un APK y lo guardará en `app/build/outputs/apk/debug/app-debug.apk`.
